### 不同的二叉树搜索树（Unique Binary Search Trees）
[原题来源：力扣（LeetCode）](https://leetcode-cn.com/problems/unique-binary-search-trees/)

#### 题目
给定一个整数 n，求以 1 ... n 为节点组成的二叉搜索树有多少种？

示例：
```
输入: 3
输出: 5
解释:
给定 n = 3, 一共有 5 种不同结构的二叉搜索树:

   1         3     3      2      1
    \       /     /      / \      \
     3     2     1      1   3      2
    /     /       \                 \
   2     1         2                 3
```

#### 思路
我将`二叉搜索树`简称为`二叉树`了，要注意两者之间的差别。

这道题我觉得比前一道题目简单一些，因为不用去生成二叉树，所以这道题相对于上一题要换一个思路，不过依旧是使用动态规划来做。

以一个数组`[1, 2, 3, 4, 5]`为例子，如果将 3 作为根结点，它的所有二叉树数量为`[1, 2]`和`[4, 5]`能够生成的二叉树的数量的乘积。

那么数组`[1, 2, 3, 4, 5]`的二叉树总数为每个结点作为根结点的所有可能二叉树的和。

#### 伪代码
```
NUM-TREES(n int)
    res = new Array()
    res[0] = 1
    res[1] = 1
    for i = 2 to n
        for j = 1 to i
            res[i] += res[j-1] * res[i-j]
    return res[n]
```
