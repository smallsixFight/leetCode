### 格雷编码（Gray Code）
[原题来源：力扣（LeetCode）](https://leetcode-cn.com/problems/gray-code/)

#### 题目
格雷编码是一个二进制数字系统，在该系统中，两个连续的数值仅有一个位数的差异。

给定一个代表编码总位数的非负整数 n，打印其格雷编码序列。格雷编码序列必须以 0 开头。

示例 1：
```
输入: 2
输出: [0,1,3,2]
解释:
00 - 0
01 - 1
11 - 3
10 - 2

对于给定的 n，其格雷编码序列并不唯一。
例如，[0,2,3,1] 也是一个有效的格雷编码序列。

00 - 0
10 - 2
11 - 3
01 - 1
```
示例 2：
```
输入: 0
输出: [0]
解释: 我们定义格雷编码序列必须以 0 开头。
     给定编码总位数为 n 的格雷编码序列，其长度为 2n。当 n = 0 时，长度为 20 = 1。
     因此，当 n = 0 时，其格雷编码序列为 [0]。
```

#### 思路
额...这道题直接查了 [wiki](https://zh.wikipedia.org/wiki/%E6%A0%BC%E9%9B%B7%E7%A0%81)，然后知道了 grayCode 的由来和计算公式，然后就根据公式直接做出来了。

额，有点无言。

#### 伪代码
```
GRAY-CODE(n)
    l = math.pow(2, n)
    res = new Array()
    for i := 0; i < l; i ++
        res.push(i ^ (i >> 1))
    return res
```
