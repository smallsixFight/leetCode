### 矩阵置零（Set Matrix Zeroes）
[原题来源：力扣（LeetCode）](https://leetcode-cn.com/problems/set-matrix-zeroes/)

#### 题目
给定一个 m x n 的矩阵，如果一个元素为 0，则将其所在行和列的所有元素都设为 0。请使用原地算法。

示例 1：
```
输入: 
[
  [1,1,1],
  [1,0,1],
  [1,1,1]
]
输出: 
[
  [1,0,1],
  [0,0,0],
  [1,0,1]
]
```
示例 2：
```
输入: 
[
  [0,1,2,0],
  [3,4,5,2],
  [1,3,1,5]
]
输出: 
[
  [0,0,0,0],
  [0,4,5,0],
  [0,3,1,0]
]
```

**进阶**

- 一个直接的解决方案是使用  O(mn) 的额外空间，但这并不是一个好的解决方案。
- 一个简单的改进方案是使用 O(m + n) 的额外空间，但这仍然不是最好的解决方案。
你能想出一个常数空间的解决方案吗？

#### 思路
先记录一下 O(mn) 和 O(m+n) 的思路。

O(mn) 就是直接初始化一个跟原数组相同的数组，然后对原数组进行遍历，赋上符合要求的值，然后返回。

O(m+n) 就是先对原先的数组进行遍历，将存在 0 的行和列分别存起来，然后原地修改值，因为每行和每列至多只需要记录一次，所以空间复杂度为 O(m+n)。

常数的空间复杂度思路如下：

如果从每一行和每一列的第一个元素开始遍历，那么后续中如果在该行和该列中存在 0，那么可以将每行和每列的第一个元素标记为 0，因为已经被遍历过，所以不会有后续影响。  

遍历结束后，依次对第一个元素为 0 的行和列分别设置为 0；由于使用第一行和第一列用来做标记，那么重新赋值的第一次要跳过第一行或第一列。

然后由于第一行和第一列的交点是`matrix[0][0]`，所以需要分别记录第一行和第一列是否需要置零。

#### 伪代码
```
SET-ZEROES(matrix)
    s1, s2 = 1, 1
    for i = 0 to matrix.length -1
        for j = 0 to matrix.length -1
            if matrix[i][j] == 0
                if i = 0
                    s1 = 0
                if j = 0
                    s2 = 0
                matrix[0][j] = 0
                matrix[i][0] = 0
    for i = 1 to matrix.length -1
        if matrix[i][0] == 0
            for j = 1 to matrix[0].length -1
                matrix[i][j] = 0
    for i = s2 to matrix[0].length -1
        if matrix[0][i] == 0
            for j = 1 to matrix.length -1
                matrix[j][i] = 0
    if s1 == 0
        for i = 1 to matrix[0].length -1
            matrix[0][i] = 0
```
