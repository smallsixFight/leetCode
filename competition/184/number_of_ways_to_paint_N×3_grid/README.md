#### 思路
这是一道 DP 题目，第`n`行能够使用的方案受限制取决与第`n-1`项的的方案数量。

通过观察可以发现，填充类型可以划分为`121`和`123`两种填充类型，即分别使用两种颜色和使用三种颜色进行填充。

对于使用两种颜色填充的，它的下一行有五种可选择方案，而三种颜色的有四种可选择方案。

上述的五种可选择方案中有三种是使用两种颜色填充和两种是使用三种颜色填充的；四种可选择的方案中则各占两种，分别对两种方式进行统计，可以得到：
```
p1[n]=p1[n-1]*(2/5)*5 + p2[n-1]*(2/4)*4 => p1[n] = p1[n-1]*2 + p2[n-1]*2
p2[n]=p1[n-1]*(3/5)*5 + p2[n-1]*(2/4)*4 => p1[n] = p1[n-1]*3 + p2[n-1]*2

nums = p1+p2
```
