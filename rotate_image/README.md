### 旋转图像（Rotate Image）
[原题来源：力扣（LeetCode）](https://leetcode-cn.com/problems/rotate-image/)

#### 题目
给定一个 n × n 的二维矩阵表示一个图像。

将图像顺时针旋转 90 度。

**说明**  
你必须在**原地**旋转图像，这意味着你需要直接修改输入的二维矩阵。**请不要**使用另一个矩阵来旋转图像。

示例 1：
```
给定 matrix = 
[
  [1,2,3],
  [4,5,6],
  [7,8,9]
],

原地旋转输入矩阵，使其变为:
[
  [7,4,1],
  [8,5,2],
  [9,6,3]
]
```

示例 2：
```
给定 matrix =
[
  [ 5, 1, 9,11],
  [ 2, 4, 8,10],
  [13, 3, 6, 7],
  [15,14,12,16]
], 

原地旋转输入矩阵，使其变为:
[
  [15,13, 2, 5],
  [14, 3, 4, 1],
  [12, 6, 8, 9],
  [16, 7,10,11]
]
```

#### 思路
因为题目是一个 n*n 的矩阵，所以只需要考虑正方形的情况。

可以从外层开始，依次将外层的每个节点及它旋转后的节点都完成旋转，然后对内层进行旋转。

由于旋转九十度，所以固定旋转四次即可；另外，可以找到节点`matrix[i,j]`旋转后的位置为`matrix[j, n-i-1]`。

#### 伪代码
```
ROTATE(matrix)
    n = matrix.length
    for l = 0 to n/2
        for s to l to n-l-2
            i, j = l, s
            t = matrix[i][s]
            for k to 3
                i, j = j, n-i-1
                temp = matrix[i][j]
                matrix[i][j] = t
                t = temp
```
