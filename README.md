### 前言
本项目为个人的 LeetCode 刷题记录，题目均来源于 [LeetCode](https://leetcode-cn.com/)。

### 限制
为了达到效果以及能够持之以恒，我对自己提了下列的要求限制：

- 每天做 3～4 题，周末可以在这基础上增加 1～2 题；
- 做题步骤：先简单写下思路，然后写伪代码，然后用 golang 具体实现；
- 在自己做完后看下 LeetCode 上是否有更好的思路，有的话补充加入；
- 保存运行结果。
